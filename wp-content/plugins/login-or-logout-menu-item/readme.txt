Add a dynamic "Login" or "Logout" menu item to any WordPress Menu.

== Description ==

With this plugin you can easily add a dynamic login/logout menu item to any menu on your WordPress site. The menu item will change based on whether the current user is logged in or logged out.

[How to use?](https://wordpress.org/support/topic/instructions-to-use-please/#post-9679222)

This is a simplified and updated version of the BAW Login/Logout Menu plugin by Juliobox. Thanks to him for his prior efforts!

== Installation ==

1. Download a .zip of this plugin file
1. Go to your WordPress Dashboard -> Plugins page
1. Click "Add New" -> "Upload"
1. Upload the .zip of this plugin and activate it
1. [How to use?](https://wordpress.org/support/topic/instructions-to-use-please/#post-9679222)
